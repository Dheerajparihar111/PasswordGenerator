🔐 Password Generator (Java CLI)

A secure and customizable Command Line Password Generator built using Java.
This project allows users to generate strong passwords based on selected criteria and evaluates password strength.

📌 Features

Generate password with custom length

Option to include:

Uppercase letters (A–Z)

Lowercase letters (a–z)

Numbers (0–9)

Special characters (!@#$%^&*)

Password strength checker (Weak / Medium / Strong)

Input validation

Clean OOP-based structure

🛠️ Technologies Used

Java

OOP (Object-Oriented Programming)

Array & String Handling

Regular Expressions

CLI (Command Line Interface)
